HTML Table Reference


Table Tags

Tag Name Description
<table></table> Table The wrapper element for all HTML tables.
<thead></thead> Table Head The set of rows defining the column headers in a table.
<tbody></tbody> Table Body The set of rows containing actual table data.
<tr></tr> Table Row The table row container.
<td></td> Table Data The table row container.
<tfoot></tfoot> Table Foot The set of rows defining the footer in a table.


Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
